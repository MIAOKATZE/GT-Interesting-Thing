模组说明  Mod Description
=====================

| 说明 | Description |
| --- | --- |
| 为GTNH游玩提供了一些有趣的小道具，其本身极大提升了游玩体验，同时尽量平衡使用代价。 | Provides interesting gadgets for GTNH gameplay, greatly enhancing the gaming experience while balancing usage costs. |

## 功能介绍  Features

| 功能 | Feature |
| --- | --- |
| **浮空核心**&lt;br&gt;• 装备到任意Baubles饰品栏即获得飞行能力&lt;br&gt;• 飞行时每tick消耗0.02饥饿值&lt;br&gt;• 饥饿值低于3格时自动禁用飞行 | **Float Core**&lt;br&gt;• Equip to any Baubles slot to gain flight&lt;br&gt;• Consumes 0.02 hunger per tick while flying&lt;br&gt;• Automatically disables flight when hunger below 3 points |
| **电力浮空核心**&lt;br&gt;• 装备到任意Baubles饰品栏即获得飞行能力&lt;br&gt;• LV电压等级，容量32 000 000 EU&lt;br&gt;• 飞行时每tick消耗128 EU（20tick=2560 EU）&lt;br&gt;• 电力耗尽时消耗饥饿值0.01/tick&lt;br&gt;• 饥饿值低于3格且电力不足时自动禁用飞行 | **Electric Float Core**&lt;br&gt;• Equip to any Baubles slot to gain flight&lt;br&gt;• LV tier, 32 000 000 EU capacity&lt;br&gt;• Consumes 128 EU per tick while flying (20 ticks = 2560 EU)&lt;br&gt;• Consumes 0.01 hunger per tick when electricity depleted&lt;br&gt;• Automatically disables flight when both hunger below 3 and electricity insufficient |
